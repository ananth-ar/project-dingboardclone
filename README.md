# DingBoard Clone
A simple image editor using PixiJS and vanilla TypeScript(unfinished)


## Quick Start
```bash
git clone https://github.com/ananth-ar/project-dingboardclone.git
cd project-dingboardclone/editorPixijs
npm install
npm run dev
```

## Tech Stack
- PixiJS (WebGL)
- Vanilla TypeScript
